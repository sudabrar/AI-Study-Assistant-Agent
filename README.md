# AI Study Assistant Agent

## Project Overview

This project implements an AI Study Assistant Agent using n8n and Google Gemini API.

The agent receives a user's question and generates an intelligent response using Google's Gemini Large Language Model.

## Features

* AI Agent
* Google Gemini API
* Memory Support
* Workflow Automation
* Natural Language Question Answering

## Technologies Used

* n8n
* Google Gemini API

## Workflow

User Question → AI Agent → Gemini LLM → Response

## How to Run

1. Install n8n
2. Import the provided JSON workflow
3. Configure Gemini API Key
4. Execute the workflow
5. Ask questions through the chat interface
